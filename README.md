# 포트리스 미니 🎮

이제 **방 코드**로 여러 쌍이 동시에 플레이할 수 있어요. 컴퓨터 두 대(또는 그 이상 여러 쌍)가 같은 서버에 접속해서 각자 방에서 대전합니다.

## 준비물
- Node.js (https://nodejs.org 에서 LTS 버전 설치)
- 서버를 켤 컴퓨터 1대 (계속 켜져 있어야 함) + 접속할 사람들

## 실행 방법 (서버 켜는 사람만)

```
npm install
node server.js
```

실행하면 접속 주소가 표시됩니다:
```
──────────────────────────────────────
  포트리스 미니 서버 시작! (다중 방 지원)
  이 컴퓨터에서:   http://localhost:3000
  다른 컴퓨터에서: http://192.168.0.15:3000   ← 예시
──────────────────────────────────────
```

## 플레이 방법

1. 브라우저에서 위 주소로 접속 (같은 네트워크의 다른 사람은 `192.168.x.x:3000` 주소로)
2. **"방 만들고 링크 공유하기"** 버튼 클릭 → 자동으로 4자리 방 코드가 생성되고 그 방으로 입장
3. **"🔗 링크 복사"** 버튼으로 링크를 복사해서 카톡/디스코드 등으로 상대에게 전달
4. 상대는 링크를 클릭하거나, 랜딩 화면에서 코드를 직접 입력해서 입장
5. 각자 탱크 선택 → P1이 맵 선택 → 둘 다 "준비 완료!" → 게임 시작!

**여러 쌍이 동시에** 플레이하고 싶다면, 각 쌍이 서로 다른 방(다른 코드)을 만들면 됩니다. 방은 서로 완전히 독립적이라 섞이지 않아요.

## 노션 등에 올려서 공유하기

지금처럼 내 컴퓨터에서 서버를 켜두면, 그 컴퓨터가 켜져 있고 인터넷/네트워크가 연결된 동안만 다른 사람이 접속 가능합니다. 노션에 올려서 언제든 접속되게 하려면 Render, Railway, Fly.io 같은 곳에 서버를 배포해서 24시간 켜두는 방법이 있습니다. (배포가 필요하면 말씀해주세요, 도와드릴게요!)

## 조작법
| 키 | 동작 |
|---|---|
| A / D | 좌우 이동 (연료 제한, 매 턴 충전) |
| ← / → | 포신 각도 조절 |
| 1 / 2 | 일반탄 / 강력탄 전환 |
| SPACE | 누르고 있으면 파워 충전, 떼면 발사! |

## 탱크 소개
| 탱크 | 특징 | 일반탄 (넓은 범위) | 강력탄 (좁고 강함) |
|---|---|---|---|
| 🟢 발칸 | 균형형 | 표준 포탄 | 집속 철갑탄 |
| 🔵 레인버드 | 분열형 | 정점에서 3발 분열 | 빠른 드릴 슛 |
| 🔴 매머드 | 중전차 (포탄 무거움) | 대구경 유탄 (지형 대파괴) | 벙커버스터 (최강 한 방) |
| 🟡 트윈스 | 연사형 (이동 빠름) | 더블 캐넌 2연발 | 더블 스나이프 2연발 |

## 게임 규칙
- 매 턴 바람이 랜덤하게 바뀝니다 (화면 상단 화살표 확인!)
- 포탄은 지형을 파괴합니다. 발밑이 무너지면 추락 데미지!
- 맵 밖으로 떨어지면 즉사. 상대 HP를 0으로 만들면 승리!

## 문제 해결
- **방이 가득 찼다고 나올 때**: 이미 그 방 코드로 2명이 들어가 있는 것. 새 방을 만들거나 다른 코드를 쓰세요.
- **다른 컴퓨터에서 접속이 안 될 때**: 서버 컴퓨터의 방화벽에서 Node.js를 허용해주세요.
- **혼자 테스트하고 싶을 때**: 브라우저 창(또는 시크릿 창) 2개를 열고 같은 방 링크로 둘 다 입장하면 됩니다.
