# 포트리스 미니 🎮

**방 코드**로 여러 그룹이 동시에 플레이할 수 있고, **1:1 대전**, **2:2 팀전**, 그리고 혼자서도 즐기는 **1인용 아케이드(CPU 3연전)**를 지원합니다. 맵마다 다른 8비트 배경음악도 자동 재생돼요.

## 준비물
- Node.js (https://nodejs.org 에서 LTS 버전 설치)
- 여럿이 플레이하려면: 서버를 켤 컴퓨터 1대 + 접속할 사람들
- 1인용만 즐긴다면: 그냥 내 컴퓨터에서 서버만 켜면 끝!

## 실행 방법

```
npm install
node server.js
```

실행하면 접속 주소가 표시됩니다:
```
──────────────────────────────────────
  포트리스 미니 서버 시작! (다중 방, 1:1 / 2:2 지원)
  이 컴퓨터에서:   http://localhost:3000
  다른 컴퓨터에서: http://192.168.0.15:3000   ← 예시
──────────────────────────────────────
```

## 1인용 아케이드 모드 🕹️

1. 접속 후 **"🕹️ 아케이드 (CPU 3연전)"** 클릭
2. 탱크 하나를 선택 → 나머지 3대와 랜덤 순서로 순서대로 1:1 대결 (라운드마다 랜덤 맵 + 랜덤 순서)
3. **매 라운드 체력은 100으로 새로 시작**합니다
4. 라운드가 올라갈수록 CPU 조준이 더 정확해집니다 (1라운드 관대함 → 3라운드 매서움)
5. **3연승하면** 옛날 오락실처럼 3글자 이니셜을 입력해 **명예의 전당**에 기록! (총 소요 턴수가 적을수록 상위 랭킹)
6. 랜딩 화면의 **"🏆 명예의 전당"**에서 언제든 순위표 확인 가능

명예의 전당 기록은 서버에 저장되며, 서버를 재배포하면 초기화될 수 있습니다 (무료 호스팅 플랫폼 특성).

## 함께 플레이하기 (1:1 / 2:2)

1. 브라우저에서 접속 주소로 이동
2. **게임 방식**에서 "1:1 대전" 또는 "2:2 팀전" 선택
3. **"방 만들고 링크 공유하기"** 클릭 → 방 코드 자동 생성
4. **"🔗 링크 복사"** 버튼으로 링크를 함께할 사람들에게 전달
5. **2:2는 같은 링크로 4명이 입장** — 먼저 들어온 2명이 팀A, 나중 2명이 팀B가 됩니다
6. 각자 탱크 선택 → 방장(P1)이 맵 선택 → 전원 "준비 완료!" → 게임 시작

여러 그룹이 동시에 플레이하고 싶다면, 그룹마다 다른 방(다른 코드)을 만들면 됩니다. 방은 서로 완전히 독립적이에요.

## 2:2 팀전 규칙
- 턴은 팀이 번갈아 진행됩니다 (예: P1 → P3 → P2 → P4)
- 한쪽 팀의 탱크 2대가 모두 파괴되면 상대 팀 승리
- 팀원이 죽어도 살아있는 팀원 차례에 계속 플레이 가능
- 화면 상단에 팀별 합산 체력 바가 표시됩니다

## 노션 등에 올려서 공유하기

지금처럼 내 컴퓨터에서 서버를 켜두면, 그 컴퓨터가 켜져 있는 동안만 다른 사람이 접속 가능합니다. 언제든 접속되게 하려면 Render, Railway 같은 곳에 배포해서 24시간 켜두면 됩니다.

## 조작법
| 키 | 동작 |
|---|---|
| A / D | 좌우 이동 (연료 제한, 매 턴 충전) |
| ← / → | 포신 각도 조절 |
| 1 / 2 | 일반탄 / 강력탄 전환 |
| SPACE | 누르고 있으면 파워 충전, 떼면 발사! |

## 탱크 소개
| 탱크 | 특징 | 일반탄 (넓은 범위) | 강력탄 (좁고 강함) |
|---|---|---|---|
| 🟢 발칸 | 균형형 | 표준 포탄 | 집속 철갑탄 |
| 🔵 레인버드 | 분열형 | 정점에서 3발 분열 | 빠른 드릴 슛 |
| 🔴 매머드 | 중전차 (포탄 무거움) | 대구경 유탄 (지형 대파괴) | 벙커버스터 (최강 한 방) |
| 🟡 트윈스 | 연사형 (이동 빠름) | 더블 캐넌 2연발 | 더블 스나이프 2연발 |

## 게임 규칙
- 매 턴 바람이 랜덤하게 바뀝니다 (화면 상단 화살표 확인!)
- 포탄은 지형을 파괴합니다. 발밑이 무너지면 추락 데미지!
- 맵 밖으로 떨어지면 즉사.
- 맵마다 다른 8비트 배경음악이 자동 재생됩니다 (HUD의 🔊 버튼으로 음소거 가능)

## 문제 해결
- **방이 가득 찼다고 나올 때**: 이미 그 방 코드로 정원(1:1은 2명, 2:2는 4명)이 다 찬 것. 새 방을 만들거나 다른 코드를 쓰세요.
- **다른 컴퓨터에서 접속이 안 될 때**: 서버 컴퓨터의 방화벽에서 Node.js를 허용해주세요.
- **혼자 테스트하고 싶을 때**: 브라우저 창(또는 시크릿 창) 여러 개를 열고 같은 방 링크로 입장하면 됩니다.
